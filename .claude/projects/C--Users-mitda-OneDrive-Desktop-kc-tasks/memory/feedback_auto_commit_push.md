name: Auto commit and push on feature changes
description: After any code change for a new feature, automatically git add, commit, and push to GitHub
type: feedback
---

After implementing a feature or code update, automatically git add, commit, and push to GitHub — don't wait for the user to ask.

**Why:** The app auto-deploys from GitHub via Vercel, so pushing immediately gets the live site updated.
**How to apply:** Every time the user requests a feature or code change, complete the code work then run git add, commit, and push to main without asking for confirmation.
